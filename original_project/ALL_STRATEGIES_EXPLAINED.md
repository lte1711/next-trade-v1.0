# 원본 프로젝트 전체 전략 설명

## 🎯 전략 개요

### 기본 철학:
- **단순함**: 복잡한 조건보다 효과적인 단순한 접근
- **적응성**: 시장 상태에 따른 동적 전략 선택
- **안정성**: 일관된 성과와 오류 방지

### 핵심 원리:
1. **시장 국면 분석**: BULL_MARKET, BEAR_MARKET, SIDEWAYS_MARKET
2. **전략 편향**: bullish, bearish, neutral, adaptive
3. **신호 강도**: 0.4-0.5 임계값 기반 BUY/SELL 결정

## 📊 5가지 핵심 전략

### 1. 모멘텀 전략 (momentum_strategy)

#### 원리:
- **추세 추종**: 강한 추세 방향으로 거래
- **가속도**: 가격 변화의 가속도 활용
- **심리적 요인**: 군중 심리 활용

#### 진입 조건:
```python
if market_regime["regime"] == "BULL_MARKET" and market_bias in ["bullish", "adaptive"]:
    signal_strength = 0.8  # 강한 신호
elif market_regime["regime"] == "BEAR_MARKET" and market_bias in ["bearish", "adaptive"]:
    signal_strength = 0.8  # 강한 신호
else:
    signal_strength = 0.2  # 약한 신호
```

#### 특징:
- **강점**: 추세장에서 높은 수익률
- **약점**: 횡보장에서 잦은 손실
- **적용**: 명확한 추세가 있는 시장

### 2. 평균 회귀 전략 (mean_reversion_strategy)

#### 원리:
- **균형 회귀**: 가격은 평균으로 회귀하는 경향
- **과매수/과매도**: 극단적인 가격 반전 활용
- **통계적 우위**: 장기적 평균 활용

#### 진입 조건:
```python
if market_regime["regime"] == "SIDEWAYS_MARKET":
    signal_strength = 0.7  # 강한 신호
else:
    signal_strength = 0.3  # 약한 신호
```

#### 특징:
- **강점**: 횡보장에서 안정적 수익
- **약점**: 강한 추세에서 큰 손실
- **적용**: 박스권 장세

### 3. 변동성 전략 (volatility_strategy)

#### 원리:
- **변동성 확대**: 높은 변동성에서 기회 포착
- **브레이크아웃**: 변동성 돌파 활용
- **위험 프리미엄**: 변동성에 대한 보상 받음

#### 진입 조건:
```python
if market_regime["volatility"] > 0.03:
    signal_strength = 0.8  # 강한 신호
else:
    signal_strength = 0.2  # 약한 신호
```

#### 특징:
- **강점**: 고변동성 시장에서 큰 수익
- **약점**: 저변동성에서 기회 부족
- **적용**: 뉴스 이벤트, 경제 지표 발표 시

### 4. 추세 추종 전략 (trend_following_strategy)

#### 원리:
- **추세 지속**: 현재 추세는 계속된다는 가정
- **모멘텀**: 가격 모멘텀 활용
- **시장 강도**: 추세 강도 측정

#### 진입 조건:
```python
if market_regime["strength"] > 0.02:
    signal_strength = 0.7  # 강한 신호
else:
    signal_strength = 0.3  # 약한 신호
```

#### 특징:
- **강점**: 장기 추세에서 큰 수익
- **약점**: 추세 전환 시 큰 손실
- **적용**: 장기적인 시장 방향성

### 5. 차익거래 전략 (arbitrage_strategy)

#### 원리:
- **가격 불균형**: 시장 비효율성 활용
- **동시 거래**: 매수와 매도 동시 실행
- **위험 회피**: 시장 방향성 무관한 수익

#### 진입 조건:
```python
signal_strength = 0.5  # 중립적 신호
```

#### 특징:
- **강점**: 시장 방향성 무관한 수익
- **약점**: 기회가 적고 수익률 낮음
- **적용**: 시장 비효율성이 큰 경우

## 🔄 동적 전략 생성

### 파라미터 최적화:
```python
# 시장 변동성에 따른 레버리지 조정
base_leverage = 10.0 + (market_volatility * 5)
leverage = min(max(base_leverage, 5.0), 50.0)  # 5x-50x 범위

# 동적 수익률 설정
win_rate = 0.4 + (random.random() * 0.3)  # 40%-70%
avg_return = 0.1 + (random.random() * 0.4)  # 10%-50%

# 동적 리스크 관리
risk_per_trade = 0.01 + (random.random() * 0.03)  # 1%-4%
stop_loss = 0.03 + (random.random() * 0.07)  # 3%-10%
take_profit = stop_loss * (2 + random.random())  # 손실의 2-3배
```

### 시장 편향 설정:
- **bullish**: 상승장 선호
- **bearish**: 하락장 선호
- **neutral**: 중립적 접근
- **adaptive**: 시장 상태에 따라 자동 조정
