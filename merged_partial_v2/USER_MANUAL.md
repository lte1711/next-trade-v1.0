# NEXT-TRADE V1 사용자 설치 및 실행 설명서

## 🎯 개요
본 설명서는 NEXT-TRADE V1의 설치부터 실행까지의 전체 과정을 사용자 관점에서 안내합니다.

---

## 📦 1. 설치 준비

### 시스템 요구사항
- **운영체제**: Windows 10/11 (64비트)
- **메모리**: 최소 4GB RAM (권장 8GB 이상)
- **디스크 공간**: 최소 100MB 여유 공간
- **네트워크**: 안정적인 인터넷 연결
- **권한**: 관리자 권한 (설치 시 필요)

### 사전 준비
- [ ] 바이러스 백신 일시 중지 (설치 오류 방지)
- [ ] 인터넷 연결 상태 확인
- [ ] 관리자 계정으로 로그인

---

## 🚀 2. 설치 절차

### 설치 파일 실행
1. **설치 파일 다운로드**
   - `install.exe` 파일을 다운로드합니다.

2. **설치 시작**
   - `install.exe` 파일을 마우스 오른쪽 버튼으로 클릭
   - "관리자 권한으로 실행" 선택
   - UAC 권한 요청 시 "예" 클릭

3. **설치 진행**
   - 설치 경로: `C:\tradev1`로 자동 설치
   - 설치 진행 상태 표시 확인
   - 설치 완료 메시지 대기

### 설치 완료 확인
- [ ] 바탕화면에 "Launch Merged Partial V2 v1" 바로가기 생성
- [ ] 시작 메뉴에 "Launch Merged Partial V2 v1" 등록
- [ ] `C:\tradev1` 폴더 생성 확인

---

## 🖥️ 3. 프로그램 실행

### 방법 1: 바로가기 실행
1. **바탕화면 실행**
   - "Launch Merged Partial V2 v1" 아이콘 더블클릭

2. **시작 메뉴 실행**
   - 시작 메뉴 → "Launch Merged Partial V2 v1" 클릭

### 방법 2: 대시보드 바로 실행
1. **대시보드 바로가기 실행**
   - "Open Merged Partial V2 Dashboard" 아이콘 더블클릭

### 실행 확인
- [ ] 콘솔창 없이 백그라운드에서 실행
- [ ] 약 3초 후 브라우저가 자동 열림
- [ ] 대시보드 화면 정상 표시

## 🌐 4. 대시보드 사용법

### 대시보드 접속
- **주소**: `http://127.0.0.1:8787`
- **브라우저**: Chrome, Edge, Firefox 등 최신 브라우저

### 주요 화면 구성

#### 📊 상태 카드 영역
1. **진행 상태**: 현재 프로그램 실행 상태
2. **실행 상태**: Dry Run/Live 모드 표시
3. **시장 상태**: 거래소 연결 상태
4. **최근 실행**: 최근 실행 이력
5. **자동매매 사이클**: 자동운용 현재 상태
6. **자금 요약**: 총자금/투자금/잔여금/수익률
7. **의사결정 상태**: 현재 의사결정 로직 상태
8. **최우선 추천**: 현재 추천 종목
9. **실행 결과**: 최신 실행 결과
10. **투자 종목**: 현재 보유 포지션
11. **리스크 게이트**: 리스크 관리 상태

#### 🎛️ 제어 버튼 영역
1. **새로고침**: 화면 데이터 새로고침
2. **드라이런 시작**: 시뮬레이션 모드 시작
3. **라이브 시작**: 실제 거래소 연동 모드 시작
4. **자동 중지**: 자동운용 중지
5. **헬스체크**: 시스템 상태 확인
6. **추천 실행**: 추천 종목 자동 실행
7. **전체 청산 및 중지**: 모든 포지션 청산 및 중지

## 🎮 5. 기능별 사용법

### 드라이런 모드 (초보자 권장)
- **목적**: 실제 자금 없이 전략 테스트
- **특징**: 
  - 실제 시장 데이터 사용
  - 자금 손익 없이 시뮬레이션만 진행
  - 전략 성과 확인 가능

**사용법**:
1. "드라이런 시작" 버튼 클릭
2. 상태 카드에서 "Dry Run" 모드 확인
3. 실행 결과 카드에서 시뮬레이션 결과 확인

### 라이브 모드 (고급자용)
- **목적**: 실제 자금으로 자동거래
- **특징**:
  - 실제 거래소 연동
  - 실제 자금 손익 발생
  - 자동매매 실행

**사용법**:
1. 사전에 거래소 API 키 설정
2. "라이브 시작" 버튼 클릭
3. 상태 카드에서 "Live" 모드 확인
4. 자금 요약에서 실제 손익 확인

### 추천 시스템
- **기능**: AI 기반 종목 추천
- **확인**: 최우선 추천 카드에서 확인
- **실행**: "추천 실행" 버튼으로 자동 실행

### 자금 관리
- **확인**: 자금 요약 카드에서 확인
- **정보**:
  - 총자금: 계정 총잔고
  - 투자금: 현재 투자 중인 금액
  - 잔여금: 추가 투자 가능 금액
  - 수익금: 현재까지의 총수익
  - 수익률: 백분율 수익률

### 포지션 관리
- **확인**: 투자 종목 카드에서 확인
- **정보**:
  - 종목명: 보유 중인 암호화폐
  - 투자금: 해당 종목 투자액
  - 진입가/현재가: 실시간 가격 정보
  - 수익금/수익률: 개별 종목 수익

## 🛠️ 8. 문제 해결

### 일반적인 문제

#### 1. 프로그램 실행 안됨
**원인**: 권한 문제 또는 바이러스 백신 차단
**해결**:
- 관리자 권한으로 실행
- 바이러스 백신 일시 중지
- Windows Defender 예외 등록

#### 2. 대시보드 접속 안됨
**원인**: 포트 충돌 또는 방화벽 차단
**해결**:
- `http://127.0.0.1:8787` 주소 확인
- 다른 프로그램과 포트 충돌 확인
- 방화벽에서 포트 8787 허용

#### 3. 데이터 표시 안됨
**원인**: API 연결 문제 또는 데이터 로딩 지연
**해결**:
- "새로고침" 버튼 클릭
- 인터넷 연결 상태 확인
- 브라우저 새로고침 (F5)

#### 4. 자동운용 작동 안됨
**원인**: API 설정 문제 또는 계정 문제
**해결**:
- API 키 설정 확인
- 계정 잔고 확인
- 거래소 서비스 상태 확인

### 오류 메시지별 대응

#### "API 연결 실패"
- 인터넷 연결 확인
- API 키 유효성 확인
- 거래소 서비스 상태 확인

#### "잔고 부족"
- 계정 잔고 확인
- 최소 거래 금액 확인
- 자금 입금 확인

#### "주문 실패"
- 시장 상태 확인
- 거래량 확인
- 수량 정밀도 확인
